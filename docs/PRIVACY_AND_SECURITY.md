# Privacy and Security

- Practice data is stored locally in SQLite; the application does not send it to a hosted API.
- Answer text stays out of reports by default. The learner must select the export option to include the newest answer.
- Imported packs are Zod-validated and stored as data, not executed as code.
- Do not import packs containing personal data, credentials or copyrighted interview material you cannot share.
- Back up or delete the local database using your operating system controls. Deleting the database deletes practice history.
- Report suspected vulnerabilities privately using the process in `SECURITY.md`.
